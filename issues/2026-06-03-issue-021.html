Microsoft Build 2026: MAI-Familie — sechs Modelle, eine Strategie, null OpenAI-Abhängigkeit

Build 2026. Microsoft feuert nicht nur ein Modell ab — sie feuern eine ganze Familie. Die Frage ist nicht mehr, ob sie im Rennen sind. Die Frage ist, wer jetzt hinter ihnen zurückfällt.

Microsoft kündigte auf Build 2026 sechs neue MAI-Modelle an: MAI-Code-1-Flash (Coding), MAI-Thinking-1 (Reasoning), MAI-Image-2.5, MAI-Voice-2, MAI-Transcribe-1.5, plus Frontier Tuning und Scout-Agent. Das Unternehmen etabliert damit einen vollständig eigenen AI-Stack — die April-Lockerung der OpenAI-Partnerschaft wird zur Realität.

MAI-Code-1-Flash: 85.8% Microsoft adversarial coding benchmark, ~51% SWE-Bench Pro. 60% fewer tokens on complex tasks vs comparable models

MAI-Thinking-1: Microsoft's first reasoning model. Competitive with Claude Opus 4.6 on SWE-Bench Pro

Microsoft Scout: Always-on personal agent for work — built on OpenClaw.

Frontier Tuning: Domain-specific model tuning.

April 2026: restrictions in Microsoft-OpenAI partnership lifted, giving Microsoft right to serve own models. Build 2026 is first full public exercise of that right.
